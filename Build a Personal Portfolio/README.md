#Build a Personal Portfolio  
Responsive personal portfolio website create with HTML and CSS. This project is from freecodecamp task to practice web development.  
## Features 
- Responsive Design
- Welcome section with Keyframe Animations
- Project showcase using CSS Grid
- Navigation bar with About, Myproject, and Contact
## What I Learn  
- CSS Animation
  The welcome section uses CSS keyframe to create introduction adn text effects.
- CSS Grid
  The project section uses Grid to organize my project cards, flexible and easy to maintain across the different screen sizes.
- Contact Form
  I use formspree to connect the contact form and receive message throught email.
  
## Tools  
- HTML
- CSS
- Formspree
