Front End for Begginer

Front end is part of website that users can interact and see. It uses three main tools:

  1. HTML
  2. CSS
  3. Javascript

HTML

HTML (Hyper text markup language) is a markup language for build a webpage.

CSS

CSS (Cascading Style Sheets) is a language used to style webpage so the webpage look beautifully.

Javascript

Javascript is language to make webpage look interactive
